穿越机所需的设备和工具

{% endblock %} {% block content %} 想要对穿越机进行使用、检查、维修，还需要一些设备和工具。
一些必要的物品：
  1. 焊接维修工具
  2. 遥控器
  3. 图传接收器
  4. 航模电池充电器（平衡充）
一些可选的物品：
  1. 外置高频头

焊接维修工具

必要工具：
  1. 电烙铁
  2. 六角螺丝刀
  3. 焊锡丝
  4. 万用表
  5. 8mm扳手
  6. 吸锡器或吸锡带
可选工具：
  1. 铜丝球
  2. 夹具
  3. BGA焊油
  4. 可调电源
  5. 示波器

遥控器

用于远程控制穿越机做出各种动作，详见遥控器的基本使用

图传接收器

用于观看穿越机回传的图像，详见图传接收器的基本使用

航模电池充电器（平衡充）

用于对电池充电，详见充电器的基本使用

外置高频头

用于发射无线信号远程控制穿越机。虽然遥控器已经内置了高频头，但通过连接外置高频头的方式可以使遥控器用来控制更多种类的接收机。使用方式详见遥控器的基本使用 安装高频头一节
