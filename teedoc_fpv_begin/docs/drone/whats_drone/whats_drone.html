什么是穿越机

{% endblock %} {% block content %}
穿越机是一种多旋翼飞行器，但和普通无人机有较大差别。主要体现在以下方面
  1. 穿越机结构简单，大多是由碳纤维板组成骨架，大部分零部件暴露在外。
  2. 穿越机动力强劲，载重能力远比同尺寸无人机大。
  3. 穿越机速度极快，速度可轻松达到200Km/h以上。
  4. 穿越机可以在任何状态下起飞，完全不会因地形等因素导致无法启动。
  5. 穿越机硬件自由度高，飞手可根据自己的想法自行选购或自主设计配件进行组装。
  6. 穿越机智能化程度低，操作难度高，但飞行没有姿态限制，飞手可根据需要控制它以任何姿态飞行。
  7. 穿越机一般不具备悬停、自动巡航等辅助飞行系统。
总体来说穿越机的门槛远高于开箱即用的无人机，飞手需要会看接线示意图、会焊接、会调试，而且需要经过长时间的飞行训练才能熟练掌握。 {% endblock %}
